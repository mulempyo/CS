
선형대수 심화 개념 설명 정리 (면접 대비)

동치: a이면 b이고 b이면 a이다.

1. 전치행렬(Transpose)
전치행렬은 행(row)과 열(column)을 서로 바꾼 행렬이다.
행렬 A의 전치행렬을 A^T로 표기한다.
회전행렬의 역행렬 계산, 내적 표현 등에 사용된다.

2. 행렬곱(Matrix Multiplication)
행렬곱은 한 행렬의 행과 다른 행렬의 열을 내적하여 계산한다.
좌표변환, 선형변환, 로봇의 자세 변환 등에 사용된다.

3. 행렬곱의 성질
- 결합법칙 성립: (AB)C=A(BC)
- 분배법칙 성립: A(B+C)=AB+AC
- 교환법칙 불성립: AB≠BA

4. 행렬곱의 전치
(AB)^T = B^T A^T
순서가 뒤집히는 것이 중요하다.

5. 역행렬(Inverse Matrix)
행렬 A에 대해 AA^-1=I를 만족하는 행렬.
연립방정식 해 구하기, 좌표계 변환에 사용된다.

a^-1, ab, a^T, ka, a^k가 가역일때,
(a^-1)^-1 = a
(ab)^-1 = b^-1a^-1
(a^T)^-1 = (a^-1)^T
(ka)^-1 = 1/k * (a)^-1
(a^k)^-1 = (a^-1)^k

6. 대칭행렬(Symmetric Matrix)
A^T=A
공분산 행렬이 대표적인 예이다.

7. 반대칭행렬(Skew-Symmetric Matrix)
A^T=-A
회전과 외적을 행렬 형태로 표현할 때 자주 사용된다.

8. 대각행렬(Diagonal Matrix)
대각 원소 외에는 모두 0인 행렬.
계산이 매우 간단하다.

9. 삼각행렬(Triangular Matrix)
상삼각행렬 또는 하삼각행렬.
LU분해, Cholesky 분해에 사용된다.
하(상) 삼각행렬의 곱은 하(상) 삼각행렬이다.
가역인 하(상) 삼각행렬의 역은 하(상) 삼각행렬이다.

10. 행렬식(Determinant)
행렬이 공간의 길이·면적·부피를 얼마나 변화시키는지 나타낸다.
det(A)=0이면 공간이 찌그러져 정보가 사라진다.
a11a22 - a12a21

11. 행렬식의 성질
행 교환 시 부호 반전. |b| = -|a|
삼각행렬의 행렬식은 대각원소의 곱.
det(A) = det(A^T)

12. 행렬식과 가역성
det(A)≠0 이면 역행렬 존재.
det(A)=0 이면 역행렬 존재하지 않음.

13. 곱의 행렬식
정사각행렬 A,B에 대해 det(AB)=det(A)det(B)

14. 역행렬의 행렬식
det(A^-1)=1/det(A)

15. 수반행렬(Adjoint Matrix)
여인수는 특정 원소의 행과 열을 제거해 얻은 소행렬식에 부호 (−1)i+j를 곱한 값이며, 행렬식 계산과 역행렬 계산에 사용됨
여인수행렬을 전치한 행렬.
역행렬 계산 공식에 사용된다.
A가 가역일때 A^-1 = 1/detA * adjA

16. 가역의 동치명제
A는 정사각행렬
- det(A)≠0
- rank(A)=n
- 열벡터 선형독립
- 고유값에 0이 없음

행렬식 기하학적 의미
- 면적(2차원) 또는 부피(3차원)가 얼마나 확대·축소되는지를 나타내는 값
- det > 0 -> 좌표계 방향 유지
- det < 0 -> 좌표계가 뒤집힘
- det = 0 -> 면적 또는 부피가 0: 2차원 → 선분, 3차원 → 평면으로 찌그러짐.

17. 벡터(Vector)
크기와 방향을 가진 수학적 객체.
위치, 속도, 힘 등을 표현한다.

18. 내적(Dot Product)
두 벡터의 방향 유사성을 측정.
결과는 스칼라이다.

19. 외적(Cross Product)
3차원 공간에서만 정의됨.
두 벡터에 모두 수직인 벡터 생성.
회전축 계산에 사용된다.

20. 내적과 외적의 의미
내적 → 각도, 유사도, 투영
외적 → 회전축, 법선벡터

21. n차원 벡터
2차원, 3차원뿐 아니라 일반적인 n차원 공간의 벡터.

22. 내적의 행렬 표현
벡터x,y와 정사각행렬A에 대해
Ax·y = x·A^Ty
x·Ay = A^Tx·y

23. 평면의 방정식
ax+by+cz+d=0
평면의 법선벡터(a,b,c)

24. 정사영(Projection)
벡터를 특정 방향으로 투영하는 연산.
최소제곱법의 핵심 개념이다.

25. 벡터공간(Vector Space)
벡터 덧셈과 스칼라곱에 대해 닫혀있는 공간.

유클리드공간
벡터의 합, 스칼라곱, 내적이 정의된 공간

26. 부분공간(Subspace)
벡터공간 내부에 존재하는 작은 벡터공간.

27. 일차결합(Linear Combination)
벡터들을 상수배 후 더한 것.

28. 생성집합(Spanning Set)
벡터공간 전체를 생성할 수 있는 벡터 집합.

29. 일차독립(Linear Independence)
(한 벡터가 다른 벡터의 일차결합으로 표현 안됨, 일차결합식에서 상수가 모두 0)

30. 일차종속(Linear Dependence)
(한 벡터가 다른 벡터의 일차결합으로 표현 됨, 일차결합식에서 상수가 모두 0이 아님)

31. 독립과 종속의 의미
독립 → 정보 중복 없음
종속 → 정보 중복 존재

32. 기저(Basis)
공간을 생성하는 최소 독립 벡터 집합.

33. 차원(Dimension)
기저 벡터 개수.

34. 행공간(Row Space)
행벡터들이 생성하는 공간.

35. 열공간(Column Space)
열벡터들이 생성하는 공간.

36. rank(A)=n의 의미
A는 정사각행렬
rank: 계수행렬의 계수
A의 모든 행(열) 벡터는 R^n에서 일차독립임.
행렬이 가역임.

37. 좌표벡터
특정 기저 기준으로 표현된 벡터.

38. 추이행렬(Change-of-Basis Matrix)
기저를 변경하는 행렬.

40. 직교기저(Orthogonal Basis)
기저 벡터들이 서로 직교.

41. 정규직교기저(Orthonormal Basis)
직교하면서 길이가 1.

42. QR분해
A는 정사각행렬 아니여도됨
A=QR
Q는 직교행렬, R은 가역인 상삼각행렬.
최소제곱 문제 해결에 사용.

43. 직교행렬 성질
직교행렬은 정사각행렬
Q^TQ=I
Q^T = Q^-1
|Ax| = |x|
벡터x,y에 대해 Ax·Ay = x·y 
A의 열벡터들은 정규직교집합, A의 행벡터들은 정규직교집합
길이와 각도를 보존.

44. 고유값(Eigenvalue)
변환 후 벡터가 얼마나 늘어나는지 나타냄.

고유값 기하학적 의미
선형변환 후에도 방향이 바뀌지 않는 특별한 방향에서의 확대·축소 비율

45. 고유벡터(Eigenvector)
선형변환 후 방향이 변하지 않는 벡터.

46. 고유벡터 일차독립성
서로 다른 고유값의 고유벡터는 일차독립

대칭행렬의 고유벡터
대칭행렬의 서로 다른 고유값에 대응하는 고유벡터는 직교

47. 대칭행렬의 대각화
대칭행렬은 항상 대각화 가능.

48. 직교대각화
A=QDQ^T
수치계산에서 매우 중요.

49. 선형변환(Linear Transformation)
벡터를 다른 벡터로 변환하는 함수.
행렬로 표현 가능.
임의의 벡터 x,y에 대해 변환L:V->W
L(x + y) = L(x) + L(y)
L(kx) = kL(x)

50. 선형변환 성질
L(0) = 0
L(x - y) = L(x) - L(y)

51. 핵(Kernel)
변환 결과가 0이 되는 벡터 집합.
L(x) = 0 

52. 핵공간
Kernel이 이루는 공간.

53. 치역(Image)
선형변환 결과가 도달 가능한 공간, 함수에서의 y

54. 상공간(Range)
치역과 같은 의미.

55. 선형변환 행렬
모든 선형변환은 행렬로 표현 가능.

56. 순서기저(Ordered Basis)
기저 벡터의 순서를 지정한 기저.

SLAM 연결
- 회전행렬 → 좌표변환
- 직교행렬 → SO(3)
- 정사영 → 최소제곱
- QR분해 → 최적화
- 고유값/고유벡터 → PCA
- 대칭행렬 → 공분산 행렬
- Rank → Jacobian 분석
